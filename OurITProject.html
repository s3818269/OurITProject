  1. Tools and Technologies

There are countless tools to create a livestream platform like Twitch, NimoTv, NoNo Live. Each tool has a different programming method and different algorithms that are suitable for many types of website. For our project, we decided to use Wowza Streaming Engines tools to build up the livestream app on a web platform. Wowza Streaming Engine is also known as Wowza Media Server prior to version 4. Developed and created by Wowza Media Systems.

Wowza is a unified live media server software. Use the server to stream videos and play back videos, audio on demand via IP network to transmit to computer devices such as PCs, laptops. In addition, the server is developed based on JavaScript programming. This software is asking for licenses, besides that, there will be a 30-day free trial for developers. However, this free trial version can only provide the basic functionality to create a livestream platform. But to be able to develop more deeply and diversified, this application requires developers to purchase for the licenses and get the latest Wowza version. Wowza Streaming Engine 4.8.5 is the latest authorized version for developers. In order to run and maintain the website stably, we need hardware devices able to work at the system server. We choose a workstation-oriented computer configuration which has a CPU and VGA that is strong enough to be able to work multi task functions, such as encode videos, and host a server. In order to host a server and record livestream, we use Wowza Streaming Cloud version 1.3.

  1. Testing

Secondly, we test the function of our project by broadcasting a testing livestream and see the outcome. Furthermore, to test if our program can run on different kinds of devices we create the early access version of our project and ask the streamers for testing the new platform. We plan to ask approximately 100 streamers from many regions around the world to test our product and give some feedback to us. In exchange, we will pay a certain amount of money for them. Our expected outcome is the videos have HD quality or higher and connection not being interrupted.

  1. Skills and Jobs

As a project manager, 4 positions that we need to recruit in order to develop this project are Information Architect, Web Developer, User Interface (UI) Designer and Finance Manager. Firstly for the Web Developer position, this job is in charge of coding, designing and layout a website based on the group manager’s demand. The web developer is also required to have experience in programming languages such as JavaScript, C++, HTML, database systems. Secondly, the UI Designer is responsible for creating the concept of the user experience, ensuring all interactions are intuitive and easy for customers to use, as well as improve the appearance of the software. In addition, having a skill on various visual design programs and some soft skills such as problem solving skills and teamwork is the must. Thirdly, for the Finance Manager, this job is in charge of planning the budget and providing the financial advice to the team leader. The requirements for this position are negotiation skills, good communication skills, keen eye for detail. Lastly, the Information Architect’s missions are to plan and design the information structure for websites and web applications. This job is required to have some experience in web production softwares such as Visio, Dreamweaver, Photoshop.
